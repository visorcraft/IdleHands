# `idlehands plugins`

管理 Gateway 网关插件/扩展（进程内加载）。

相关内容：

- 插件系统：[插件](/tools/plugin)
- 插件清单 + 模式：[插件清单](/plugins/manifest)
- 安全加固：[安全](/gateway/security)

## 命令

```bash
idlehands plugins list
idlehands plugins info <id>
idlehands plugins enable <id>
idlehands plugins disable <id>
idlehands plugins doctor
idlehands plugins update <id>
idlehands plugins update --all
```

内置插件随 IdleHands 一起发布，但默认禁用。使用 `plugins enable` 来激活它们。

所有插件必须提供 `idlehands.plugin.json` 文件，其中包含内联 JSON Schema（`configSchema`，即使为空）。缺少或无效的清单或模式会阻止插件加载并导致配置验证失败。

### 安装

```bash
idlehands plugins install <path-or-spec>
```

安全提示：将插件安装视为运行代码。优先使用固定版本。

支持的归档格式：`.zip`、`.tgz`、`.tar.gz`、`.tar`。

使用 `--link` 避免复制本地目录（添加到 `plugins.load.paths`）：

```bash
idlehands plugins install -l ./my-plugin
```

### 更新

```bash
idlehands plugins update <id>
idlehands plugins update --all
idlehands plugins update <id> --dry-run
```

更新仅适用于从 npm 安装的插件（在 `plugins.installs` 中跟踪）。
