# `idlehands tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
idlehands tui
idlehands tui --url ws://127.0.0.1:18789 --token <token>
idlehands tui --session main --deliver
```
