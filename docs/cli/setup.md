# `idlehands setup`

Initialize `~/.idlehands/idlehands.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
idlehands setup
idlehands setup --workspace ~/.idlehands/workspace
```

To run the wizard via setup:

```bash
idlehands setup --wizard
```
